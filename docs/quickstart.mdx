## Prerequisites

- Python 3.10–3.13
- An API key from a supported provider (e.g. [OpenAI](https://developers.openai.com/api/docs), [Anthropic](https://platform.claude.com/docs/en/home), or [Gemini](https://ai.google.dev/gemini-api/docs/api-key))

## Get started

<Steps>
  <Step title="Install Arksim">
    ```bash
    pip install arksim
    ```
  </Step>
  <Step title="Set your API key">
    Set the key for the provider your config uses (e.g. OpenAI):

    ```bash
    export OPENAI_API_KEY="your-api-key"
    ```

    For other providers:

    ```bash
    export ANTHROPIC_API_KEY="your-api-key"
    export GEMINI_API_KEY="your-api-key"
    ```
  </Step>
  <Step title="Run Simulation & Evaluation">
    Download the example projects (scenarios and configs included), then run simulation and evaluation via the CLI, Python, or the web UI.

    **Download examples:** From your project directory, run:

    ```bash
    arksim examples
    ```

    This creates an `examples/` folder with three projects: **bank-insurance**, **e-commerce**, and **openclaw**. To list or download a single example: `arksim examples --list`, then `arksim examples <name>`.

    <Tabs>
      <Tab title="CLI">
        From the directory of the example you want to run, use that example's config (e.g. `config.yaml`). Replace `<example>` with one of: `bank-insurance`, `e-commerce`, `openclaw`.

        ```bash
        cd examples/<example>
        arksim simulate-evaluate config.yaml
        ```

        Example for the bank-insurance quick path (OpenAI only, no local server):

        ```bash
        cd examples/bank-insurance
        arksim simulate-evaluate config.yaml
        ```

        Ensure the API key required by that example's config is set from Step 2 (e.g. `OPENAI_API_KEY` for bank-insurance and e-commerce `config.yaml`).

        <Tip>
          To learn what goes in a config (agent, scenario path, outputs), see [Scenarios](/build-scenario), [Simulation](/simulate-conversation), and [Evaluation](/evaluate-conversation). For e-commerce or bank-insurance with a local RAG agent, see [E-commerce customer service](/e-commerce-customer-service-agent-evaluation) and [Insurance customer service](/insurance-customer-service-agent-evaluation).
        </Tip>
      </Tab>
      <Tab title="Python">
        Run simulation and evaluation from Python. Use paths under the example you downloaded; the snippet below uses `bank-insurance`—change the example directory name to use `e-commerce` or `openclaw`.

        ```python
        from arksim.simulation_engine import run_simulation, SimulationInput
        from arksim.evaluator import run_evaluation, EvaluationInput
        from arksim.config import AgentConfig

        # Example agent config (OpenAI; match the example's config.yaml)
        agent_config = AgentConfig(
            agent_type="chat_completions",
            agent_name="bank-insurance",  # or e-commerce, openclaw
            api_config={
                "endpoint": "https://api.openai.com/v1/chat/completions",
                "headers": {
                    "Content-Type": "application/json",
                    "Authorization": "Bearer ${OPENAI_API_KEY}",
                },
                "body": {
                    "model": "gpt-5.1",
                    "messages": [
                        {"role": "system", "content": "You are a customer service chatbot for XYZ Bank insurance."}
                    ],
                },
            },
        )

        simulation = run_simulation(SimulationInput(
            agent_config=agent_config,
            scenario_file_path="examples/bank-insurance/scenarios.json",  # or e-commerce, openclaw
            num_conversations_per_scenario=1,
            max_turns=5,
            num_workers="auto",
            output_file_path="./simulation.json",
        ))

        evaluation = run_evaluation(EvaluationInput(
            simulation_file_path="./simulation.json",
            scenario_file_path="examples/bank-insurance/scenarios.json",  # or e-commerce, openclaw
            output_dir="./evaluation",
            model="gpt-5.1",
            provider="openai",
            num_workers="auto",
            generate_html_report=True,
        ))

        print("Simulation and evaluation complete.")
        print(f"Simulated {len(simulation.conversations)} conversations.")
        print("Evaluation results saved to ./evaluation")
        ```
      </Tab>
      <Tab title="UI">
        Run simulation and evaluation from your browser with the arksim web UI.

        ```bash
        arksim ui
        ```

        Your browser opens at [**http://localhost:8080**](http://localhost:8080). Set your API key in the environment before starting (e.g. `OPENAI_API_KEY`). Then open or create a config that points at one of your downloaded examples (e.g. `examples/bank-insurance/config.yaml`).
      </Tab>
    </Tabs>
  </Step>
  <Step title="View Results">
    - **Conversations:** Open `conversations.json` in your simulation output directory for full transcripts.
    - **Evaluation:** Open `final_report.html` or `final_report.md` and the CSV files in your evaluation output directory for metrics and failure analysis.

    Use these to see how your agent performed and where it failed or could be improved.
  </Step>
</Steps>
